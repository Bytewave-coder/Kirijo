# Next.js to Android APK Setup

This package contains:
- Capacitor config
- Static export Next.js config
- GitHub Actions APK build workflow
- Android build instructions

## How to Use

1. Copy all files into your Next.js project.
2. Push project to GitHub.
3. GitHub Actions will automatically build APKs.
4. Download APK from Actions artifacts.

## Important

Your Next.js app should support:
- Static export
- Client-side rendering where possible

## GitHub Actions

After pushing:
- Open GitHub repository
- Go to Actions tab
- Wait for workflow
- Download APK artifact
